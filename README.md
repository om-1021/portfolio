# Om Dubey — Portfolio

Personal portfolio website. Single-page, no build step — just static HTML/CSS/JS.

## Files
- `index.html` — the entire site (styles + scripts inline)
- `om.png` — profile photo

## Local preview
Open `index.html` in any browser.

## Photo
The About section loads `om.png`. The black-and-white + lime-tint look is applied automatically via CSS, so any color photo works.

## Deploy
Free hosting via Netlify or Vercel — see `DEPLOY.md`. To publish with GitHub Pages: push this folder to a repo, then Settings → Pages → deploy from the `main` branch root.
